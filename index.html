Javascript中的面向对象

博学谷 第2天

分支

  1. 创建分支

    git branch 分支名称
    git checkout -b 分支名称

  2. 切换分支

    git checkout 分支名称

  3. 删除分支

    git branch -d 分支名称

  4. 合并分支

    git merge 分支名称

  5. 后悔药

    git log -g

    git branch 分支名称 版本号

远程仓库

共享版本库（源代码）

  1. 创建

    git init –bare

  2. 给远程仓库中推送内容

    git push 远程仓库地址 本地分支名称:远程分支名称

  3. 获取远程仓库中所有内容

    git clone 远程仓库地址 [文件夹名称]

  4. 获取远程仓库中更新了的内容

    git pull 远程仓库地址 分支名称

  5. 注意 在push之前一定要先pull

远程地址的别名设置

  1. 添加别名

    git remote add 别名 远程仓库地址

  2. 查看所有别名

    git remote

  3. 删除别名

    git remote remove 别名

  4. 重命名别名

    git remote rename 旧名 新名

  5. 获取别名对应的地址

    git remote get-url 别名

  6. 设置别名对应的地址

    git remote set-url 别名 远程仓库地址

github的免密登录设置

  1. 生成秘钥对（一个电脑执行一次就行）
    ssh-keygen -t rsa
一路回车，冒泡就成功
  2. 在github的设置中 添加一个ssh key
  3. key中的内容就是将生成的公钥内容粘贴进去就可以了
  4. 以后使用github的时候，直接（只有）使用ssh连接就可以实现免密

github静态页面

  1. 在github中创建一个仓库
  2. 仓库的名字必须是 用户名.github.io
  3. 将写好的静态页面push到这个cnagku
  4. 通过 用户名.github.io 这个地址就可以访问网站了

注意： 一个账户只能有一个这样的项目！！！
